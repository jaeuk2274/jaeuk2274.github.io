현재 포커스 실행
- Ctrl + Shift + R

이전 실행 (상단 셀렉트 박스 기준)
- Ctrl + R

라인 복사(현재줄)
- Command + D

라인 삭제(현재줄)
- Command + 백스페이스

라인 합치기(현재 포커스 줄로 합침)
- Ctrl + Shift + J 

라인 단위로 옮김(구문상 오류 없는 한) 
- Shift + Command + 상/하 방향키 

라인 단위로 옮김(구문상 오류 상관없음) 
- Shift + Option + 상/하 방향키
