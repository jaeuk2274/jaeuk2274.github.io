## 독립적으로 실행 가능한 JAR

스프링 부트 애플리케이션. 


mvn clean - target 파일 삭제
mvn package - jar 하나 생성
mvn clean package - 같이 써 줘도 된다.

mvn package -DskipTests -- 테스트 스킵

이 jar 파일 안에 모든게 다 들어있다.
target
+- app
  +- BOOT-INF 
    +- classes - 내가 만든 java class 파일들
    +- lib - 이 안에 라이브러리 다 들어있다.
  +- META-INF
  +- org
...
